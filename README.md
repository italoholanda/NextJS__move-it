# Executando

```bash
# Clone o Repositório
$ git@github.com:tavareshenrique/moveit-nlw.git
```

```bash
# Acesse a pasta do projeto e baixe as dependências
$ yarn
```

```bash
# Execute
$ yarn dev
```

Acesse <http://localhost:3000> para ver o resultado.
